API to get Indonesia stock exchange daily data from Pasardana website.

1. List of Stocks

https://indonesia-stock-exchange.vercel.app/api

Query Parameter

Parameter Description
search Type: string
Search query
page Type: int | Default: 1
Page to retrieve
per_page Type: int | Default: 25
Number of records per page
sort_by Type: string | Default: Code
Sort the results by the given field. Possible values: Code, Name, SectorName, SubSectorName, Last, AdjustedClosingPrice, PrevClosingPrice, AdjustedOpenPrice, AdjustedHighPrice, AdjustedLowPrice, Per, Pbr, Volume, Value, Frequency, OneDay, OneWeek, OneMonth, ThreeMonth, SixMonth, OneYear, ThreeYear, FiveYear, TenYear, Mtd, Ytd, Capitalization, BetaOneYear, StdevOneYear, LastDate,
sort_direction Type: string | Default: asc
Sort direction: asc or desc.
index_id Type: int | Default:
Filter stocks by the given index id. Get all possible index ids from the List Indices endpoint.
sector_id Type: int | Default:
Filter stocks by the given sector id. Get all possible sector ids from the List Sectors endpoint.
subsector_id Type: int | Default:
Filter stocks by the given subsector id. Get all possible subsector ids from the List Sectors endpoint.
